## About Me

I build small, practical tools around cybersecurity, monitoring, OSINT, and clean technical interfaces.

My work usually sits between defensive analysis, network behavior, Domain Name System intelligence, and lightweight utilities that solve annoying daily problems.

I care about tools that feel fast, clear, and useful — not overloaded, not noisy, and not pretending to be bigger than they are.

## What I Like Building

* Monitoring dashboards with sharp dark interfaces
* Small Windows tools that save time
* OSINT-style analysis flows
* Domain and network activity analyzers
* Lightweight web pages without heavy frameworks
* Practical utilities that stay quiet until needed
